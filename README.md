# Projeto Hídrico: Nanobolhas em São Paulo

Este é um projeto que apresenta uma proposta inovadora para o tratamento e revitalização das águas urbanas, industriais e para reúso em São Paulo, aplicando tecnologia de nanobolhas.

## Tecnologias Utilizadas

- HTML5
- TailwindCSS (via CDN)
- Chart.js
- JavaScript Vanilla

## Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/[seu-usuario]/Projeto-geo.git
cd Projeto-geo
```

2. Instale as dependências:
```bash
npm install
```

3. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

4. Abra o navegador em `http://localhost:3000`

## Deploy

Este projeto está configurado para deploy automático no Vercel.

1. Fork este repositório
2. Conecte ao Vercel
3. Deploy!
